# jpyc_faucet_backend

JPYC Faucetのバックエンドです。

## Project setup
```
npm install
```
.env_sampleファイルの中身を記入して

.env.localと.env.prodにリネームしてください

### Compiles and hot-reloads for development
```
npm run start:local
```
